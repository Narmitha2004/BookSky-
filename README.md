BookSky is a simple and user-friendly book management web application developed 
using HTML, CSS, and JavaScript. The project allows users to add, view, and delete books 
dynamically through an interactive UI. It is designed to help beginners understand 
core JavaScript concepts such as DOM manipulation, event handling, and dynamic content rendering.

Features

Add books with title, author, and description
Display books in a clean card-based layout
Delete books instantl
Popup form for better user experience
Responsive and minimal UI design

Technologies Used
HTML5 – Structure of the application
CSS3 – Styling and layout
JavaScript (ES6) – Functionality and DOM manipulation

PURPOSE:
This project was built to strengthen JavaScript fundamentals and gain hands-on experience in building interactive frontend applications without using any external libraries or frameworks.
